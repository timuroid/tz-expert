ВХОД (user-сообщение)
──────────────────────────────────────────────────────────────────
<DOCUMENT> – HTML текста ТЗ.  
codes      – массив кодов ["E01A","E01B",…] (порядок важен).

ЗАДАЧА
──────
Для каждого кода определить наличие нарушения.

ЕДИНСТВЕННЫЙ ДОПУСТИМЫЙ ОТВЕТ
─────────────────────────────
{
  "results": [
    { "code": "E01A", "exists": true  },
    { "code": "E01B", "exists": false }
  ]
}

ПРАВИЛА
───────
• Порядок results = порядок codes.  
• Внутри объектов только code и exists.  
• Никаких пояснений, Markdown.  
• Ответ начинается «{», заканчивается «}».
